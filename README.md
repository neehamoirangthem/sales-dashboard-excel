# Sales Dashboard (Excel Project)
## Overview
This project is an interactive sales dashboard built using Microsoft Excel. It analyzes sales performance, customer behavior and regional trends.

## Features
- KPI cards for Total Sales, Orders and Profit.
- Top 5 customers analysis.
- Sales by category and profit by region.
- Key business insights.

## Tools Used
- Microsoft Excel
- Pivot Tables
- XLOOKUP
- INDEX-MATCH
- Data Visualization

## Key Insights
- Technology category contributes the highest sales.
- West region delivers strong profit performance.
- Top customers generate a significant portion  of revenue.
